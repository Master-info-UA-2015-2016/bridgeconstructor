bridgeconstructor

Programme d'intelligence artificiel pour choisir le type, les matériaux et le prix pour la construction de ponts

Modélisation Algorithmique :

  • Une classe Bridge : les caractéristique du pont, définies selon les règles appliquées
  • Une classe Environment : faits décris par l'utilisateur avant l'exécution de la recherche du pont à construire
  • Une classe AIEngine : chargées d'appliquer les règles sur la Base de faits afin d'en déduire les ponts constructibles, et parmi ceux-ci, le meilleur choix (surement le moins cher).
